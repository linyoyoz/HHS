# 房地產查詢系統
主要功能:登入、註冊、查詢
# 系統必備

Ruby 版本 2.5.3
Rail 版本 5.2.2
mysql 版本 5.7.24(資料庫主要資料)

#房地產資料來源
https://price.housefun.com.tw
由此網站查詢高雄各地區之每一坪的平均房價，並整合成共15區的房價資料
